# Wave Dataset

> The **Wave dataset** is a synthetic 1-D regression dataset generated by `mglearn.datasets.make_wave(n_samples)`. A single continuous input `x` is paired with a target `y` that follows a sinusoidal-with-noise shape. It serves as the chapter's go-to teaching example for [[Binning]], [[Polynomial Features]], [[Polynomial Regression]], and [[Interaction Feature|bin × slope]] demonstrations because its 1-D nature lets every transformation be plotted and inspected directly ([[Introduction to Machine Learning with Python (Muller & Guido)]]).

## Key Contributions

- Single input feature — lets every feature-engineering step be visualised as a line plot.
- Demonstrates that linear regression's single-slope fit is qualitatively wrong on non-monotonic data.
- After [[Binning]] into 10 buckets + one-hot encoding, linear regression produces a piecewise-constant fit indistinguishable from a shallow decision tree ([[Introduction to Machine Learning with Python (Muller & Guido)]]).
- After adding bin × x [[Interaction Feature|interactions]], linear regression produces a piecewise-linear fit with per-bin slopes.
- Degree-10 polynomial expansion produces a smooth curve that approximates the underlying sinusoidal shape — but oscillates violently outside the training range.
