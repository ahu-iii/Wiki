# WMT 2014

The WMT 2014 datasets are standard [[Neural Machine Translation]] benchmarks from the 2014 Workshop on Machine Translation. Two language pairs are widely reported:

- **English–German (EN-DE)**: ~4.5M sentence pairs. [[Attention Is All You Need]] uses a shared source–target byte-pair vocabulary of ~37K tokens (Source: [[Attention Is All You Need]], Section 5.1).
- **English–French (EN-FR)**: ~36M sentence pairs, tokenized with a 32K word-piece vocabulary.

Results are reported as [[BLEU Score|BLEU]] on the **newstest2014** test set, with **newstest2013** used as the development set.

## Key Contributions

- Provides the canonical training and evaluation split for EN-DE and EN-FR NMT research in the mid-2010s.

## Connections

- Evaluation target for: [[Transformer]], GNMT, ConvS2S, ByteNet.
- Used with: [[BLEU Score]], [[Byte-Pair Encoding]].
